# Medicine Recommendation System Documentation

This folder contains comprehensive documentation for the MediRec project.

## Documents Included

### 1. Project_Documentation.md
**Comprehensive Project Documentation**
- Complete project overview and technical specifications
- Team information and responsibilities
- System architecture and implementation details
- Features, testing, and performance analysis
- Future enhancements and conclusions
- **Format:** Markdown (can be converted to Word using Pandoc)
- **Length:** ~15,000 words

### 2. Presentation_Slides.md
**PowerPoint Presentation Content**
- 22 detailed slides for academic presentation
- Covers all aspects from problem statement to conclusion
- Includes technical details, demonstrations, and Q&A
- **Format:** Markdown (can be converted to PowerPoint)
- **Duration:** 20-25 minutes presentation

### 3. Research_Paper.md
**Academic Research Paper**
- Formal research paper in academic format
- Abstract, literature review, methodology, results, discussion
- Comprehensive analysis with references and citations
- **Format:** Academic paper format
- **Length:** ~8,500 words
- **Sections:** 6 main sections with subsections

## How to Use These Documents

### Converting to Word Document
```bash
# Install Pandoc (if not already installed)
# Windows: Download from https://pandoc.org/installing.html

# Convert to Word document
pandoc Project_Documentation.md -o Project_Documentation.docx

# Or convert to PDF
pandoc Project_Documentation.md -o Project_Documentation.pdf
```

### Converting to PowerPoint
```bash
# Convert presentation to PowerPoint
pandoc Presentation_Slides.md -o Presentation_Slides.pptx

# Or use online converters like:
# - https://pandoc.org/try/
# - Copy content to PowerPoint manually for better formatting
```

### Research Paper Formatting
```bash
# Convert to academic paper format
pandoc Research_Paper.md -o Research_Paper.pdf --template=academic

# Or submit to academic journals in LaTeX format
pandoc Research_Paper.md -o Research_Paper.tex
```
