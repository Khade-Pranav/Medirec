About MediRec

Project Overview

MediRec is an intelligent Medicine Recommendation System developed as a Computer Engineering Project (CEP) for 2nd Year, 3rd Semester. This system uses Machine Learning algorithms to predict diseases based on user-input symptoms and provides comprehensive medical recommendations including medications, diet plans, workout routines, and precautionary measures.

Key Features:

  • AI-powered disease prediction using Support Vector Machine (SVM)
  • Smart symptom autocomplete with 132 medical symptoms
  • Comprehensive medication recommendations
  • Personalized diet and workout suggestions
  • Search history tracking and management
  • PDF report generation for medical records
  • External resource integration (Wikipedia, YouTube, PharmEasy)
  • Dark/Light theme toggle
